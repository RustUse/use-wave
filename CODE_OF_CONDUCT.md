# Code of Conduct

RustUse expects collaboration to stay technical, respectful, and constructive.

## Expected behavior

- Focus feedback on code, docs, tests, and project direction.
- Keep review comments actionable and grounded in the code or documented policy.
- Respect different experience levels and explain constraints when rejecting an approach.

## Unacceptable behavior

- Harassment, intimidation, or personal attacks.
- Discriminatory or exclusionary behavior.
- Publishing private information without permission.

## Reporting

Report conduct concerns privately to ferris@rustuse.org.

Do not use this channel for security vulnerabilities. Follow [SECURITY.md](SECURITY.md) for private vulnerability reporting.
